# Anugal

Enterprise Identity Governance and Administration Platform

## ROI Calculator

This repository includes a comprehensive ROI (Return on Investment) calculator for the Anugal platform.

### Quick Start

1. Open `roi-calculator/index.html` in a web browser
2. Fill in your organization's details
3. Click "CALCULATE ROI" to see estimated returns
4. Download a detailed report

### Features

- **Responsive Design**: Works on all devices
- **Real-time Calculations**: Instant ROI estimates
- **Detailed Reports**: Download CSV reports with all metrics
- **Multi-currency Support**: EUR/USD based on region
